# Calisma Alani

Bu proje, TypeScript kullanarak geliştirilmiş bir uygulamadır. Aşağıda projenin dosya yapısı ve her bir dosyanın işlevi hakkında bilgiler bulunmaktadır.

## Dosya Yapısı

```
calisma-alani
├── src
│   ├── app.ts          # Uygulamanın giriş noktası
│   └── types
│       └── index.ts    # Özel türler ve arayüzler
├── package.json        # NPM yapılandırma dosyası
├── tsconfig.json       # TypeScript yapılandırma dosyası
└── README.md           # Proje belgeleri
```

## Kurulum

Projenin çalışması için gerekli bağımlılıkları yüklemek için aşağıdaki komutu çalıştırın:

```
npm install
```

## Kullanım

Uygulamayı başlatmak için aşağıdaki komutu kullanın:

```
npm start
```
